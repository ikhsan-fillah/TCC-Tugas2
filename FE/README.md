# Tugas 2 - Frontend Notes (Vanilla)

## Menjalankan frontend

Buka file `index.html` langsung di browser, atau gunakan static server sederhana.

## Fitur

- Tambah catatan
- Lihat daftar catatan
- Edit catatan
- Hapus catatan

## Catatan

Pastikan backend berjalan di `http://localhost:3000` agar request API berhasil.
